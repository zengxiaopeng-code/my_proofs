How to check it yourself

The repository is public. To verify with the Lean kernel:

git clone https://github.com/zengxiaopeng-code/my_proofs
cd my_proofs && lake exe cache get && lake build
lake env lean MyProofs/PosteriorProcess/Audit.lean
